# Getting Started with Rocket

Welcome to Rocket, the platform that enables you to effortlessly create powerful backend APIs without the need for coding. This guide will walk you through the initial steps to begin your journey with Rocket.

## Sign-up and Registration

1. **Visit Our Website**: To get started, go to our website at [https://rocketapi.net](https://rocketapi.net/).

2. **Sign Up**: Click the "Get Started" button to create your Rocket account. You'll need to provide your email address, choose a secure password, and verify your email.

3. **Pricing**: By default, Rocket automatically selects the Bronze plan (free plan). To explore each plan and its pricing details, click [here](https://rocketapi.net/#pricing).

4. **Log In**: After verifying your email, log in to your Rocket account using your credentials.

## Dashboard Overview

Once you've successfully logged in, you'll arrive at your Rocket dashboard. Here's a quick overview of what you'll find:

- **Navigation**: On the left side, you'll see all the navigation buttons.
- **Top Bar**: In the top-right corner of the page, you can change the selected database. At any given time, you can only select one database.
- **Databases**: In the main section of the page, you'll find a list of all your databases.

## Navigation Links

- **Databases**: Provides a list of databases along with usage information.
- **Tables**: Lists all the tables for the selected database.
- **Data**: Allows you to add new data to selected tables.
- **APIs**: Displays a list of all generated APIs and can be used for documenting APIs.

- **Help**: Opens the help page.
- **Account**: Checks account details.
- **Subscription**: Shows all available subscriptions and can be used for upgrading to other subscriptions.
- **Logout**: Logs you out of your account.

## System Requirements

Before you start creating APIs with Rocket, make sure your development environment meets these requirements:

- **Web Browser**: Rocket is browser-based and supports the latest versions of popular browsers such as Chrome, Firefox, Safari, and Edge.

That's it! You're now ready to dive into creating your first API with Rocket. Follow the subsequent sections in this documentation to learn how to model your data, configure endpoints, and unleash the full potential of Rocket.

If you encounter any issues or have questions along the way, don't hesitate to raise a ticket [here](https://github.com/imdeepmind/Rocket/discussions).

Let's get started building your API the Rocket way!
