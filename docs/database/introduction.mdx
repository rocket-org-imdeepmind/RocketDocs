# Introduction to Databases

Within the realm of Rocket, databases serve as the cornerstone of projects, housing a comprehensive array of components ranging from tables and data to APIs and their corresponding documentation.

Consider the scenario where we aim to construct a backend system for managing movie-related data. The initial step involves establishing a database named MoviesDB. For detailed guidance on creating databases, please refer to documentation on [Creating Databases](https://help.rocketapi.net/get-started/creating-database).
