# Editing Data

Editing any added data is as simple as adding new data. Follow the procedure outlined below to edit any data on Rocket.

1. **Navigate to Your Database**: Log in to your Rocket account and select the desired database from the database selector located at the top left corner of the interface.

2. **Access Data Page**: On the left navigation menu, click on the "Data" link to access the Data page.

3. **Select Desired Table**: From the list of tables, choose the table where you want to add data. Click on the respective table to open it.

4. **Pick the data to edit**: Pick the row present in the data table to edit. Then click on the edit button present on the left side of the table.

5. **Edit Data**: You will be directed to the Edit Data page, where you'll find all the data fields based on the defined attributes of the selected table. Edit the fields based on the requirement.

6. **Click on Submit**: After editing all the needed data fields, click on the "Submit" button to save the data entry.

By following these steps, you can seamlessly add data to your Rocket database.
