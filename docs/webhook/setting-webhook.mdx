# Setting Webhook

To setup the webhooks for any API, follow the steps below.

1. **Navigate to Your Database**: Log in to your Rocket account and select the desired database from the database selector located at the top left corner of the interface.

2. **Access Webhook Page**: On the left navigation menu, click on the "Webhook" link to access the Webhook page.

3. **Select the Desired API**: From the list of APIs, select the API you want to setup the webhook for. Then click on the edit button present on the left side of the table.

4. **Setup Webhook**: You will be directed to the Edit Webhook page, where you'll find the switch to enable webhook and webhook URL field. Enable the webhook and paste the webhook URL.

5. **Click on Update**: After providing all the necessary data fields, click on the "Update" button to save the webhook.

By following these steps, you can seamlessly setup webhook for any APIs.
